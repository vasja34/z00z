# z00z_telemetry

📌 `z00z_telemetry` contains the workspace telemetry facade and the crate-level entry point for shared observability concerns.

📌 The crate currently stays intentionally small so other Z00Z components can depend on one stable place for telemetry wiring without duplicating package boundaries.

## Scope

- structured logging integration points
- future metrics and tracing adapters
- a stable crate surface for workspace observability code
