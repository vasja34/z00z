<action>
Move the concrete native backend from `src/db/redb_wallet_store/**` to
one-level `src/redb_store/*.rs` files and preserve the existing public path
through `src/db/mod.rs`, using facade wiring such as
`#[path = "../redb_store/mod.rs"] pub mod redb_wallet_store;`. Update all
internal imports, re-exports, and anchor-sensitive tests so downstream code can
continue to use `db::redb_wallet_store::*` without learning the new physical
layout. Carry the schema and wallet-guide anchors with the move or rewrite
their include paths atomically, including external source-inspection tests that
currently read `../src/db/redb_wallet_store/...` directly. This ownership also
includes the `TODO-Wallet-idea.md` and `z00z_core` asset-config source anchors
loaded by `test_mod.rs`. The RedB schema YAML belongs in the canonical non-`src/`
schema home under `crates/z00z_wallets/schemas/`, with test anchors updated in
the same slice if the move happens here.
</action>
