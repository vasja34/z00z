- Skipping the audit would let target-only vocabulary appear as live code and
  create false implementation confidence.
- Treating current rights support as finished would miss delegation,
  consumption, revocation, expiry, challenge, fee boundary, and value-rejection
  coverage.
- Treating vouchers as asset metadata or wallet payloads would create a second
  semantic authority outside settlement storage.
</threat_model>
