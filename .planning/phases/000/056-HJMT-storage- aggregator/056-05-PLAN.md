# 056-05 Plan: YAML Materialization And Startup Preflight

<objective>
Turn the Phase 056 runtime contract into real config-driven behavior and
enforce one fail-closed startup gate.

Purpose: prevent hidden constants, partial topology state, or stale lineage
from becoming accepted startup conditions.
Output: YAML-backed runtime config behavior, config digests, and explicit
preflight reject classes.
</objective>

  drift
