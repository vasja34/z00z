# 057-01 Plan: Root Generation, Shard Leaf, And Checkpoint Publication Contracts

<objective>
Freeze the executable publication contract for Phase 057 on the live storage,
aggregator, and node seams.

Purpose: make root-generation transitions, canonical `ShardRootLeafV1`, and
canonical `CheckpointPublicationV1` real byte-level contracts before any join,
transfer, or validator logic is allowed to build on them.
Output: explicit generation handling, canonical bytes and digests, monotonicity
guards, and compatibility evidence across the first public publication bridge.
</objective>
