## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| profile object -> wallet open | Missing or partial cutover would keep Snapshot as live metadata authority. |
| YAML config -> runtime defaults | Silent fallback to literals would contradict the new config contract. |
| backup/recovery defaults -> live flows | Hardcoded paths and gap width would make restore behavior differ from configured expectations. |

| T-047-08 | D | runtime defaults | mitigate | Route live defaults through YAML/env resolution and fail closed on invalid config. |
| T-047-09 | R | backup/recovery config | mitigate | Replace hardcoded values on live paths and add regression tests. |
